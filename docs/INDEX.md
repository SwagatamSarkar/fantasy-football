### .gitignore
**Git ignore patterns for the project**

- **What it is:** Configuration file for Git
- **Purpose:** Tell Git which files to ignore
- **Covers:** Maven, IDE, OS, build files
- **When needed:** Copy to project root before git operations
- **Important patterns:**
  - `target/` (Maven builds)
  - `.idea/` (IntelliJ)
  - `.vscode/` (VS Code)
  - `*.class`, `*.jar` (compiled files)
  - `.DS_Store` (Mac)
  - `Thumbs.db` (Windows)

**Use it for:** Copy to root directory, no editing needed
